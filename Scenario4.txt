*Titre : Modifier une page 

*Objectif : Cas d’utilisation permet à un acteur de modifier une page.

*Acteur : internaute / éditeur 

*Précondition :    Application installée en local 
		   Page modifiable 	

*Enchainements :
		    C1 :  L’acteur exécute le système 	 
    		    C2 : L’acteur remplit le champ de recherche
		    C3 : L’acteur clique sur le bouton ‘chercher’
		    C4 : Le système ouvre une page en locale 
		    C5 : L’acteur clique sur modifier 
		    C6 : L’acteur modifie les données 
		    C7 : L’acteur clique sur update 	
		    C8 : Le système push les modifications
