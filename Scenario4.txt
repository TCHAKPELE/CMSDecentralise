*Titre : Modifier une page 

*Objectif : Cas d’utilisation permet à un acteur de modifier une page.

*Acteur : internaute / éditeur 

*Précondition :    Application installée en local 
		   Page modifiable 	

*Enchainements :
		    1 :  L’acteur exécute le système 	 
    		    2 : L’acteur remplit le champ de recherche
		    3 : L’acteur clique sur le bouton ‘chercher’
		    4 : Le système ouvre une page en locale 
		    5 : L’acteur clique sur modifier 
		    6 : L’acteur modifie les données 
		    7 : L’acteur clique sur update 	
		    8 : Le système push les modifications
